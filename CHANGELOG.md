## [1.6.0] - 2023-07-27

- adds prettier formatting options and option to just generate entities

### Added

- `prettierConfig` key to config - Use a specific prettier config file
- `usePrettier` key to config - Use prettier to format the files
- `entitiesOnly` key to config - Export only the generated entities

### Changed
- `prettier` peer modules
